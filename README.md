# Latteee

## Project Description

Latteee is a coffee website built using Angular. It showcases a dynamic and interactive user interface for exploring coffee categories, beverages, and more.&#x20;

---

## How to Install and Run the Project

Follow these steps to set up and run the project on your local machine:

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd Latteee
   ```

2. **Install dependencies:**
   Make sure you have Node.js installed. Then, run:

   ```bash
   npm install
   ```

3. **Run the mock server:**
   Start the mock backend using `json-server`:

   ```bash
   npx json-server --watch db.json --port 3000
   ```

4. **Run the Angular development server:**
   Start the Angular application:

   ```bash
   ng serve
   ```

5. **Access the application:**
   Open your browser and navigate to:

   ```
   http://localhost:4200
   ```

---

## How to Use the Project

1. **Explore Categories:** Navigate through the different categories of beverages, such as drinks, food, and toppings.
2. **View Product Details:** Click on a category to view the products available under it.
3. **Responsive Design:** Enjoy the interactive and user-friendly interface designed to provide a seamless browsing experience.
